我的Google Chrome浏览器安装的插件扩展
发表于 2019-04-11| 更新于 2019-05-08|日志
阅读量:|评论数:

Google Chrome常用插件做个记录备忘，以免像前几年一次重置账号后忘记需要哪些插件了。
其中 “ 净化淘宝链接” 这个插件是我开发的，做电商的应该能用得上，欢迎捧场!

神器：

1. 扩展管理器Extension Manager
这个扩展可以一键开关其他扩展，简直是神器。

默认开启的扩展：

1. Adblock Plus
2. Block Site
3. Checher Plus for Gmail
4. Clear Cache
5. Google Translate
6. Google Keep
7. IDM integration Module
8. LastPass
9. Momentum
10. Search by Image
11. 净化淘宝链接
12. 浮图秀
13. 轻淘客旗舰版
14. 为什么你们就是不能加个空格呢

有需要的时候才开启的扩展：

1. Chrome Better History
2. Chrono 下载管理器
3. Chrome Cleaner Pro
4. ColorZilla
5. Earth View from Googe Earth
6. Fatkun图片批量下载
7. Format Link
8. GNOME Shell integration
9. Google Arts&Culture
10. HTTP/2 and SPDY indicator
11. Markdown Viewer
12. Octotree
13. Office Online
14. Proxy SwitchyOmega
15. Reader View
16. TamperMonkey
17. Trackr
18. Unsplash Instant
19. User-Agent Switcher for Chrome
20. Video Downloader professional
21. WasteNoTime
22. WhatFont
23. WhatRuns
24. FireShot
25. 有道云笔记网页剪报
26. 违禁词检查
27. 钦定字体
