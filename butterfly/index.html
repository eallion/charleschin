Hexo butterfly 修改备忘

记录备忘，不到24小时就踩了好多坑了。

用 git subtree 管理主题

git subtree add --prefix=themes/butterfly https://github.com/eallion/hexo-theme-butterfly.git master
git subtree push --prefix=themes/butterfly https://github.com/eallion/hexo-theme-butterfly.git master
git subtree pull --prefix=themes/butterfly https://github.com/eallion/hexo-theme-butterfly.git master

Typecho 转 hexo

先用 Tp2MD 插件导出.md格式的 Typecho 的文章，再用 Abbrink 插件转换 slug url 。

用原来的 Url 格式，奖_config.yml中设置permalink: :abbrlink/。

Ubuntu 下批量替换文章内内容：

cd ~/hexo/source/_posts
sed -i 's/slug: /abbrlink: /g' *.md

新建文章模板

vim ~/hexo/scaffolds/post.md
---
layout: post
title: {{ title }}
date: {{ date }}
abbrlink:
author: 大大的小蜗牛
status: publish
comments: true
categories: 
  - 日志
tags: 
  - hexo
---

  • themes/butterfly/layout/includes/mixins/article-sort.pug
mixin articleSort(posts)
  .article-sort
    - var year
    - posts.each(function (article) {
      - var tempYear = date(article.date, 'YYYY')
      if tempYear !== year
        - year = tempYear
        .article-sort-item.year= year
      .article-sort-item
        .article-sort-img

        .article-sort-post
          a.article-sort-item__post(href=url_for(article.path))
            i.fa.fa-clock-o(aria-hidden="true")
            time.article-sort-item__time= date(article.date)
            .article-sort-item__title= article.title || 'No Title'
    - })
